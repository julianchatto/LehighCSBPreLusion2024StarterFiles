Display

                    

                        
                    

                

                

                    

                        

                            
Block

                            
Block is the default display type. Each element is treat as its own block and each block is placed after each other in a column. Centering and positioning is notoriously difficult. 

                            

                                

                                

                            

                        

                        

                            
Inline Block

                            
Inline block allows for puttin HTML elements in a row. They can also be put inline with text as that is essentially how they're being treated in HTML. This is also difficult to center and position. 

                            

                                

                                

                            

                        

                        

                            
Flex

                            
Flex boxes make organizing HTML elements easy. The Justify Content and Align Items allow for simple centering and other positional attributes while accounting for variable pages sizes.

Functions

                    

                        
                    

                

                

                    

                        

                            

                                There are two ways to declare functions in JavaScript. A more traditional, older way and a cleaner modern way.
                            

                        

                        

                            

                                
Old Way

                                

                                    function functionName(parameterName){

                                        ...

                                    }
                                

                            

                            

                                
New Way

                                

                                    functionName = (parameterName) => {

                                        ...

                                    }
                                

                            

                        

                    

                

            

            

                
 
                    
Variables

                    

                        
                    

                

                

                    

                        

                            

                                JavaScript is a dynamically typed language. This means all variables are declared with let and const.
                            

                        

                        

                            

                                
Constant (never changes)

                                

                                    const variableName1 = 0

                                    const variableName2 = "test"

                                    const variableName3 = new Array()

                                

                                
Let (changes)

                                

                                    let variableName1 = 0

                                    let variableName2 = "test"

                                    let variableName3 = new Array()

                                

                            

                        

                    

                

            

            

                
 
                    
IDs

                    

                        
                    

                

                

                    

                        

                            Much like CSS, javascript uses identifiers to identify HTML elements. JavaScript specific identifiers are called IDs are one to one (an ID cannot be replicated). In JavaScript the HTML is refered to as the document or DOM and using the DOM one can get HTML elements in many different ways.
                        

                        
ID

                        
document.getElementById("idName")

                        
Class


                        
document.getElementsByClassName("className")

                        
Query Selector


                        
document.querySelector("div")
 
                    

                

            

            

                
 
                    
Modifying DOM

                    

                        
                    

                

                

                    

                        

                            Modifying the DOM means changing styles, text, and even HTML elements. Everything is refered to based on parent, child, and sibling elements which can make it hard to understand what is actually being modified.
                        

                        

                            <div> <--- Parent

                                <div> <--- Selected Element

                                    <p>text</p> <--- Child

                                </div>

                                <p>text</p> <--- Sibling

                            </div>
                        

                    

                    

                        

                            
Modify All Children

                            
element.innerHTML = ...

                        

                        

                            
Adding children

                            
element.appendChild(...)

                        

                        

                            
Modify Style

                            
element.style.backgroundColor = ...
